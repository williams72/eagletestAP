1.4.4 Builds Management

Under a certain version, Project managers or Project members can upload App build packages and set properties (public / private) that whether an App build package is visible for other members. This function is unified management and storage of App, easy to re-use App. As shown below:

Builds Management

1.4.4.1 Upload Build

Users in the build management page, you can click on the "add build" upload App build package. The file format of the build is determined by the project type. If the type is Android, only the .apk file is allowed. If the type is IOS, only the .ipa file is allowed.
