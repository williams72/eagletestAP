Introduction

MTC (Mobile Testing Center), the industry leading mobile application testing public cloud service platform managed by Baidu, provides solutions for the cost, technology and efficiency issues faced by developers in mobile application testing. EagleTest box is based on Baidu MTC core technology to achieve the mobile application testing. Developers and testers can use real mobile devices for automated testing and real-world debugging. Connects to the server through a USB hub, each cloud platform supports up to 60 mobile devices and can be expanded.
