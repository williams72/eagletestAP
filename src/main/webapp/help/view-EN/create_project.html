2.2 Create Project

The scenario for creating project is done through two business operations: 1 and 2. The specific process is shown as below:

Flow chart for Create Project

(1) Add User

a) The administrator log in to the system and click to the "User".

b) Click the "Add User" button to enter the user's basic information, including the user name, password and email address. Section "User" for details.

(2) Create Project

a) The administrator log in to the system and click to the "Project".

b) Click "Add Project" and enter the project information, including the project name, project type, project manager and project description. Refer to Section "Project".
