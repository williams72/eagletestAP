1.4.1 Dashboard

The dashboard is mainly able to view the test success rate in real time, and visual display more project information. The platform mainly displays three types of information, including version overview, mobile overview and test pass rate.

(1) Version Overview

The version overview is the proportion of the unreleased version in the total number of versions. In version management, version exists in three states: development, testing, and released. Please refer to section Version Management for details. The unreleased version is the number of the two states in development and testing.

(2) Mobile Overview

The mobile overview is the percentage of the total number of devices in which the free mobile is counted. Depending on the type of project (Android and IOS), the mobile is different (Android and IOS). The mobile has two states, free and busy. This mobile is busy when the mobile tests a task or is remoted to debug. The mobile status is free when the device has no test and no debugging.

(3) Test Pass Rate

The test pass rate is a statistical analysis of different versions for the compatibility test, automated script testing and traversal testing, visualizing the display of different versions of the stability.
